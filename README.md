# Realm of the Glorious Lich King

Simple roguelike in python + libtcod.



### Controls

Movement:

```
 7  8  9  or  arrow keys
  \ | /
 4     6
  / | \
 1  2  3
```

General:

```
 Alt + Enter  toggle full screen
 Ctrl + Q     close game
```

Wizard mode:

```
 F1           toggle walking through walls
 F2           toggle seeing through walls
 F12          generate new level
```
